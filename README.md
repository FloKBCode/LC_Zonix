# Zonix - Jeu multijoueur en C

Projet TP n3 - B1 Informatique - Ynov Paris 2026

## Equipe
- Florence — reseau, serveur, threads
- Marly    — rendu SDL2, menus, interface
- Sarah    — logique de jeu, flood fill, collisions

## Lancer le jeu
1. aller dans le dossier : cd paperio
2. compiler : make all
3. lancer le serveur : ./server
4. lancer les clients : ./client
